# Super cinema
> *Greatest cinema service in the universe!

## How to setup local environment

- Install NodeJS and NPM
- Install MongoDB
    > https://www.mongodb.com/download-center#community
- Clone the repository
- Run `npm install`
- Run `npm run webpack`
- Run `npm run start`
- Visit the site on `localhost:8080`

## Technologies used

- ReactJS
- PureCSS
- Webpack
- SASS
- MongoDB
- Redux
- ExpressJS
